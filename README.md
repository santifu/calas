## Com publicar a GitHub Pages

1. Crea un repositori nou a github.com (p.ex. `calas-facilitadors`)
2. Puja tots els fitxers mantenint l'estructura de carpetes anterior
3. Ves a **Settings → Pages → Branch → main → / (root)** → Save
4. La web estarà disponible a `https://<usuari>.github.io/calas-facilitadors/`

> ⚠️ Els noms de fitxer i carpeta han de ser **exactament** iguals (majúscules, accents, espais inclosos). GitHub Pages és sensible a les majúscules.
